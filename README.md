# TP — Introduction au fonctionnel

**Auteur : John DINH**

Ce projet a été réalisé dans le cadre du cours **Paradigme de Projet**.  
Il a pour objectif d’introduire les **bases de la programmation fonctionnelle** en TypeScript à travers une série d’exercices pratiques.

Le code illustre différentes notions clés :
- les **fonctions pures** et **impures** ;
- les **fonctions d’ordre supérieur** (qui prennent d’autres fonctions en paramètre) ;
- les principales **fonctions de tableau** (`map`, `filter`, `reduce`, `find`, `some`, `every`, `includes`, `flatMap`, `sort`, `slice`) ;
- la manipulation de tableaux d’objets et le **traitement fonctionnel de données**.

---

## Lancement du projet

1. **Copie le projet** sur ton ordinateur ou clone le dépôt GitHub.  
2. **Ouvre le dossier** du projet avec **Visual Studio Code** (ou ton éditeur préféré).  
3. **Ouvre un terminal (Git Bash)** dans le dossier du projet.  
4. **Exécute les commandes suivantes** pour compiler et lancer le code TypeScript : 

```bash
tsc tp-fonctionnel.ts
node tp-fonctionnel.js
```
